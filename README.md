# Argent Bank

## Contexte
Argent Bank est une nouvelle banque en ligne et souhaite percer dans le secteur bancaire

## Objectifs
- Afficher les données du backend sur l'interface via des appels API
- Implémenter la gestion des données avec Redux pour assurer le fonctionnement du front
- A l'aide de l'API , gérer la connexion déconnexion et la récupération des données de l'utilisateur

  
## Langages utilisées

- SASS
- React
- Redux

  

Projet réalisé dans le cadre de la formation Intégrateur Web d'Openclassrooms


This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
